Thư Viện Hợp Âm

Tổng hợp hợp âm cá nhân, xem nhanh trên mọi thiết bị.

Danh sách bài hát

  • Bài Demo

Hướng dẫn thêm bài mới

  1. Tạo file mới trong thư mục /songs/, ví dụ bai-moi.html.
  2. Sao chép template từ /songs/bai-demo.html.
  3. Dán lời từ Word hoặc trình soạn thảo, giữ nguyên định dạng [Am] cho hợp âm.
  4. Commit và push lên GitHub.

Deploy bằng GitHub Pages

  1. Push toàn bộ mã nguồn lên branch main (hoặc master).
  2. Trong repository GitHub, mở Settings > Pages.
  3. Chọn Source là Deploy from a branch, chọn branch main và folder / (root).
  4. Lưu lại. Sau vài phút, GitHub Pages cung cấp URL để truy cập site.

Static site - không cần backend hay build.
